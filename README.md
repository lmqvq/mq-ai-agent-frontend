# LMQ-AI健身教练 - 智能健身管理平台

一个基于Vue 3和AI技术的现代化健身管理平台，为用户提供个性化的健身指导、数据追踪和智能推荐服务。

## 🌟 项目特色

- 🤖 **AI智能教练** - 基于人工智能的个性化健身指导
- 📊 **数据可视化** - 全面的健身数据追踪和分析
- 🎯 **个性化计划** - AI定制的专属训练计划
- 📚 **知识库系统** - 丰富的健身知识和动作指导
- 🔐 **完整认证** - 安全的用户登录注册系统
- 📱 **响应式设计** - 完美适配桌面端和移动端

## 🛠️ 技术栈

### 前端技术
- **Vue 3** - 现代化的前端框架
- **Composition API** - 更好的逻辑复用
- **Vue Router** - 单页面应用路由
- **Arco Design Vue** - 企业级UI组件库
- **Sass/SCSS** - CSS预处理器

### 状态管理
- **Pinia/Vuex** - 状态管理
- **LocalStorage** - 本地数据持久化
- **Session管理** - 用户认证状态

### 开发工具
- **Vite/Webpack** - 构建工具
- **ESLint** - 代码规范检查
- **Prettier** - 代码格式化

## 📦 项目结构

```
src/
├── components/          # 公共组件
├── views/              # 页面组件
│   ├── Home.vue        # 首页
│   ├── Login.vue       # 登录页
│   ├── Register.vue    # 注册页
│   ├── UserProfile.vue # 个人中心
│   ├── FitnessData.vue # 健身数据
│   ├── FitnessPlans.vue # 健身计划
│   ├── FitnessKnowledge.vue # 健身知识库
│   ├── FitnessMaster.vue # AI健身教练
│   └── SuperAgent.vue  # AI超级智能体
├── stores/             # 状态管理
│   └── user.js         # 用户状态管理
├── services/           # API服务
│   └── api.js          # API接口封装
├── router/             # 路由配置
└── styles/             # 样式文件
```

## 🚀 快速开始

### 环境要求
- Node.js >= 14.0.0
- npm >= 6.0.0

### 安装依赖
```bash
npm install
```

### 开发环境运行
```bash
npm run serve
```

### 生产环境构建
```bash
npm run build
```

### 代码检查和修复
```bash
npm run lint
```

## 🔧 配置说明

### API配置
在 `src/services/api.js` 中配置后端API地址：
```javascript
// 本地开发地址
baseURL: 'http://localhost:8123/api',
// 线上地址
// baseURL: 'https://cozi.chat/api',
```

### 路由配置
项目使用Vue Router进行路由管理，支持：
- 页面路由跳转
- 路由守卫
- 动态路由参数

## 📱 功能演示

### 用户认证流程
1. 用户访问首页，点击登录按钮
2. 跳转到登录页面，输入账号密码
3. 登录成功后显示用户头像和信息
4. 点击用户头像显示用户菜单
5. 可以访问个人中心、退出登录等功能

### 健身数据管理
1. 在个人中心记录身体数据
2. 在健身数据页面查看数据趋势
3. 添加运动记录和卡路里消耗
4. 通过图表分析健身进展

### AI健身指导
1. 与AI健身教练对话获取建议
2. 根据个人情况生成训练计划
3. 学习健身知识和动作要领
4. 获得营养饮食指导
